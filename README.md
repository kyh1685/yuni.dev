# Yuni.dev
> Next.js, TypeScript, Tailwindcss 기반의 정적 블로그입니다.

## 실행
```shell
npm install
npm run start
```

## 스택
- Language `TypeScript`
- Framework `Next.js`
- Content `Contentlayer`
- Style `Tailwindcss`
- Deploy `Vercel`
- VCS `Git`
